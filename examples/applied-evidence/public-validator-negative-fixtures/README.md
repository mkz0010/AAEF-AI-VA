# Public Validator Negative Fixtures

This directory contains synthetic, public-safe, static negative fixtures for the public example structural validator.

The fixtures are intentionally invalid. They are used only to confirm that the read-only public validator and local test harness fail closed for known structural, linkage, hygiene, claim-boundary, and runtime-boundary defects.

These fixtures do not authorize runtime execution, scanner execution, Docker execution, credential injection, customer-target operation, or delivery.
