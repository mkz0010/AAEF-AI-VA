# Tool Gateway Example Contract Set

This directory contains fictitious examples for the MVP Tool Gateway contract flow.

## Example Flows

### allowed-action

A safe ZAP authenticated crawl request that is authorized and completed.

### denied-action

A nuclei safe template scan request that is denied because external discovery is not allowed.

### human-approval-required

An Nmap service version detection request that requires human approval and is not automatically executed.

## Rule

These examples must not contain raw credentials, real customer targets, real scan artifacts, or real vulnerability data.
