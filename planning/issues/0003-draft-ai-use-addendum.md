# 0003: Draft AI Use Addendum

## Status

Open

## Priority

P1

## Type

Legal / Policy

## Background

外部AI利用を顧客に説明し、NDA・機密情報・再委託・個人情報の論点を整理する必要がある。

## Acceptance Criteria

- AI use purpose is defined
- External AI use is disclosed
- Sensitive data exclusion is defined
- Human review requirement is defined
- Customer opt-out option is described
- No-training / retention expectations are described
