# ADR-0167: Add v0.6.97 applied evidence handback material drafting or submission decision

Status: Accepted
Date: 2026-05-08

## Context

v0.6.96 reviewed and closed the internal narrow public-safe AAEF main handback material preparation candidate.

The project should decide whether to proceed toward exact public-safe AAEF main handback text drafting planning, direct submission, pause, or another Applied Evidence gap.

## Decision

Add v0.6.97 as an Applied Evidence handback material drafting or submission decision checkpoint.

This checkpoint selects narrow public-safe AAEF main handback text drafting planning as the next checkpoint, while explicitly not preparing handback material, writing handback text, creating submittable text, creating a handback package, creating a handback draft, or opening/drafting any AAEF main issue, pull request, release note, or document change.

## Consequences

Future work should use a separate text drafting planning checkpoint before any exact public-safe AAEF main handback text is drafted.

No AAEF main work is started by this checkpoint.
