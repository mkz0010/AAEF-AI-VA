# ADR-0234: Add v0.6.164 Buyer-Facing Commercial Inquiry Boundary candidate

Status: Accepted
Date: 2026-05-12

## Context

v0.6.163 selected Buyer-Facing Commercial Inquiry Boundary as a Medium-risk work item and assigned two checkpoints.

## Decision

Add a claim-safe Buyer-Facing Commercial Inquiry Boundary candidate.

Use email-based inquiry as the selected contact model, but do not commit a public email address in this candidate because no maintainer-approved public email address has been provided.

## Consequences

v0.6.165 should review the commercial inquiry boundary and decide whether to close the work item.

No customer PoC approval, commercial contract creation, commercial license terms creation, paid engagement approval, runtime execution approval, scanner execution approval, Docker execution approval, credential injection approval, customer target approval, delivery approval, AAEF main issue, or AAEF main PR is created by this candidate.
