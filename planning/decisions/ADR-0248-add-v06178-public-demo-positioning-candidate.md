# ADR-0248: Add v0.6.178 Public Demo Positioning candidate

Status: Accepted
Date: 2026-05-12

## Context

v0.6.177 selected Public Demo Positioning as a Medium-risk work item and assigned two checkpoints.

## Decision

Add a draft Public Demo Positioning candidate.

The positioning explains how public demonstrations should show control boundaries and evidence traceability without implying live scanner activity, runtime authorization, customer PoC approval, production deployment, or diagnostic completeness.

## Consequences

v0.6.179 should review the Public Demo Positioning candidate and decide whether to close the work item.

This candidate does not create a demo artifact, start runtime/scanner work, authorize a customer PoC, modify AAEF main, or publish AAEF main contact information.
