# ADR-0222: Add v0.6.152 Control Matrix candidate

Status: Accepted
Date: 2026-05-11

## Context

v0.6.151 selected Control Matrix as a Medium-risk work item and assigned two checkpoints.

## Decision

Add a claim-safe Control Matrix candidate.

The matrix maps reviewer questions, control boundaries, expected evidence, related artifacts, explicit non-goals, and reviewer notes across the Enterprise Review Guide, Technical Due Diligence Summary, Safe PoC Boundary Template, and related test families.

## Consequences

v0.6.153 should review the matrix and decide whether to close the work item.

No customer PoC approval, commercial contract creation, runtime execution approval, scanner execution approval, Docker execution approval, credential injection approval, customer target approval, delivery approval, AAEF main issue, or AAEF main PR is created by this candidate.
