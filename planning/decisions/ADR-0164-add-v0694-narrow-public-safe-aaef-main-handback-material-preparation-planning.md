# ADR-0164: Add v0.6.94 narrow public-safe AAEF main handback material preparation planning

Status: Accepted
Date: 2026-05-08

## Context

v0.6.93 selected narrow public-safe AAEF main handback material preparation planning as the next checkpoint.

The project also needs to preserve the public/private two-layer boundary: public materials can act as trust proof and evaluation package, while implementation and adoption package details remain paid or NDA material.

## Decision

Add v0.6.94 as a narrow public-safe AAEF main handback material preparation planning checkpoint.

This checkpoint plans target material shape, two-layer public/private publication boundaries, allowed sources, forbidden sources, public-safe wording families, and review gates for a future internal material preparation candidate.

It does not prepare AAEF main handback material, open or draft AAEF main issue or PR content, write AAEF main handback text, create a handback package, create a handback draft, create final handback text, or promote any AAEF Core/Profile/Practical Package content.

## Consequences

Future work may create a narrow internal material preparation candidate only if it preserves public-safe evidence/interface boundaries and excludes paid/NDA adoption package material.

No AAEF main work is started by this checkpoint.
