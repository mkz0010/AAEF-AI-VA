# ADR-0165: Add v0.6.95 narrow public-safe AAEF main handback material preparation candidate

Status: Accepted
Date: 2026-05-08

## Context

v0.6.94 planned controls for a future narrow internal public-safe AAEF main handback material preparation candidate and incorporated the two-layer public/private publication boundary.

The project can now create an internal material preparation candidate that remains public-safe and evidence/interface-level without preparing AAEF main handback material.

## Decision

Add v0.6.95 as a narrow public-safe AAEF main handback material preparation candidate.

This checkpoint records an internal material preparation candidate and reviewer aid for later close-readiness review.

It does not prepare AAEF main handback material, open or draft AAEF main issue or PR content, write final AAEF main handback text, create a handback package, create a handback draft, or promote any AAEF Core/Profile/Practical Package content.

## Consequences

Future work should review this internal material preparation candidate and decide whether it is close-ready before any handback material or AAEF main work is considered.

No AAEF main work is started by this checkpoint.
