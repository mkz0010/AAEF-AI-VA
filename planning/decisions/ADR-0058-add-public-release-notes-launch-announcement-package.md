# ADR-0058: Add public release notes and launch announcement package

Status: Accepted  
Date: 2026-05-03

## Context

AAEF-AI-VA is now a public GitHub repository with public launch state recorded in
v0.4.4.

The next step is to prepare public communication material that explains the project
without overstating runtime readiness, production readiness, compliance status, or
authorization to scan third-party systems.

## Decision

Add a v0.4.5 public release notes and launch announcement package.

The package includes:

- GitHub Release notes draft,
- short introduction,
- longer public announcement,
- LinkedIn-style announcement draft,
- Zenn/Qiita-style technical article lead,
- commercial inquiry wording,
- security reporting wording,
- claims to use,
- claims to avoid.

## Consequences

Positive:

- Public messaging becomes reusable and consistent.
- Launch communication remains aligned with the authority-boundary model.
- Commercial inquiry wording is separated from security reporting and runtime authorization.

Negative / deferred:

- This does not publish the announcement.
- This does not create a GitHub Release.
- This does not finalize commercial contract terms.
- This does not authorize runtime execution.
