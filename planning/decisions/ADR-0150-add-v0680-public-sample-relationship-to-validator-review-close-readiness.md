# ADR-0150: Add v0.6.80 public sample relationship-to-validator review and close-readiness

Status: Accepted  
Date: 2026-05-08

## Context

v0.6.79 created a narrow documentation-only public sample relationship-to-validator candidate.

Before moving to another Applied Evidence gap, the relationship candidate should be reviewed and closed.

## Decision

Add v0.6.80 as a public sample relationship-to-validator review and close-readiness checkpoint.

This checkpoint retains the v0.6.79 relationship candidate and records it as close-ready.

It does not modify validator behavior, add validator output, create an output contract, add metadata fields, rewrite fixtures, add JSON Schema, add fixtures, start Applied Evidence implementation work, generate packages, create private review records, promote public samples, refine public sample content, change public example text, create a new walkthrough, prepare AAEF handback material, start evidence-interface handback readiness planning, or authorize runtime/scanner/Docker/credential/customer/delivery activity.

## Consequences

The public sample relationship-to-validator candidate is closed.

Future work should use a separate next-gap selection checkpoint before evidence-interface handback readiness planning, package work, public sample refinement, validator implementation, or runtime work is considered.
