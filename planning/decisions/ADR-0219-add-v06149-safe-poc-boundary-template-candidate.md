# ADR-0219: Add v0.6.149 Safe PoC Boundary Template candidate

Status: Accepted
Date: 2026-05-11

## Context

v0.6.148 selected Safe PoC Boundary Template as a Medium-risk work item and assigned two checkpoints.

## Decision

Add a claim-safe Safe PoC Boundary Template candidate for future controlled PoC boundary planning.

The template defines the information that would be required before a separately approved controlled PoC could be considered, including written authorization, roles, target scope, exclusions, time window, tool/action limits, AI request/gate boundary, credential/data handling, evidence retention, human review, stop conditions, communications, deliverables, commercial/license boundary, and claim boundaries.

## Consequences

v0.6.150 should review the template and decide whether to close the work item.

No customer PoC, commercial contract, runtime execution, scanner execution, Docker execution, credential injection, customer target, delivery authorization, AAEF main issue, or AAEF main PR is authorized by this candidate.
