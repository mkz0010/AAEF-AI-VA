# ADR-0191: Add v0.6.121 next-direction selection using risk-tiered granularity

Status: Accepted
Date: 2026-05-10

## Context

v0.6.120 adopted risk-tiered checkpoint granularity and proposed selecting future work by explicitly assigning risk tier and checkpoint count.

## Decision

Select README current/latest baseline clarity as the next work item.

Classify the selected work item as Medium risk because it is public-facing documentation-only work. Use two checkpoints: candidate preparation, then review and decision.

This v0.6.121 decision itself is a Low-risk direction-selection record and is completed in one checkpoint.

## Consequences

v0.6.122 should prepare the README current/latest baseline clarity candidate. No README wording is changed by this v0.6.121 checkpoint.
