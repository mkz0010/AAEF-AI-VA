# ADR-0286: Add v0.6.216 Public Exposure Hygiene Plan Candidate

Status: Candidate recorded

## Context

v0.6.215 selected `public_exposure_hygiene_plan` as the next high-risk work item after an external review raised public-surface and commercial-boundary concerns.

The project now needs a plan candidate before any public-facing cleanup, contact route changes, pricing/business-plan moves, README rewrite, or documentation curation occurs.

## Decision

Record v0.6.216 as the Public Exposure Hygiene Plan Candidate.

The candidate plan covers:

- README front-page value proposition and reader path
- public contact route hygiene
- pricing and commercial draft exposure hygiene
- business plan exposure hygiene
- current/latest version clarity
- external-facing candidate and draft label hygiene
- public documentation curation
- demo and motion-evidence positioning
- future cleanup sequencing
- future cleanup definition of done

## Review boundary

The candidate plan is not accepted by this ADR.

v0.6.217 must review the candidate plan and record whether it is accepted, revised, or rejected.

## Consequences

The repository gains a reviewable public exposure hygiene plan candidate without applying cleanup.

No contact route is removed, no pricing or business-plan material is moved, no README front page is rewritten, and no historical docs are deleted by this checkpoint.

Runtime demo remains necessary but deferred.

Publication remains deferred.
