# ADR-0160: Add v0.6.90 narrow public-safe AAEF main handback drafting planning

Status: Accepted
Date: 2026-05-08

## Context

v0.6.89 selected narrow public-safe AAEF main handback drafting planning as the next checkpoint.

The project should plan drafting controls before any internal handback drafting candidate, handback draft, handback package, or AAEF main work is created.

## Decision

Add v0.6.90 as a narrow public-safe AAEF main handback drafting planning checkpoint.

This checkpoint plans target audience, target artifact shape, permitted wording families, forbidden wording families, source boundaries, and review gates for a future internal drafting candidate.

It does not prepare AAEF main handback material, open or draft AAEF main issue or PR content, write AAEF main handback text, create a handback package, create a handback draft, create a handback drafting candidate, or promote any AAEF Core/Profile/Practical Package content.

## Consequences

Future work may create a narrow internal drafting candidate only if it preserves public-safe evidence/interface boundaries.

No AAEF main work is started by this checkpoint.
