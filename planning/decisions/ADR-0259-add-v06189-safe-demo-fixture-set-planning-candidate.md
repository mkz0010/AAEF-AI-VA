# ADR-0259: Add v0.6.189 Safe Demo Fixture Set Planning candidate

Status: Accepted
Date: 2026-05-13

## Context

v0.6.188 selected Safe Demo Fixture Set Planning as a Medium-risk work item and assigned two checkpoints. v0.6.187 accepted the Safe Demo Artifact Planning work item.

## Decision

Add a draft Safe Demo Fixture Set Planning candidate.

The plan defines the fixture set that could later support the accepted Safe Demo Artifact Planning document and the accepted Blocked Tool Action Request Review scenario, including fixture inventory, fixture file boundaries, request/gate/non-execution/evidence fixture boundaries, reviewer walkthrough boundary, validation expectations, file naming expectations, and public/private distinctions.

## Consequences

v0.6.190 should review the Safe Demo Fixture Set Planning candidate and decide whether to close the work item.

This candidate does not create fixtures, public samples, schemas, validators, executable demos, runtime behavior, scanner behavior, customer PoC materials, or AAEF main changes.
