# ADR-0235: Add v0.6.165 Buyer-Facing Commercial Inquiry Boundary review and decision

Status: Accepted
Date: 2026-05-12

## Context

v0.6.164 created the Buyer-Facing Commercial Inquiry Boundary candidate as checkpoint 1 of 2 for the Medium-risk work item selected in v0.6.163.

The maintainer has identified an interim AAEF-wide inquiry address outside this repository change.

## Decision

Review and accept the v0.6.164 Buyer-Facing Commercial Inquiry Boundary candidate.

The Medium-risk work item is closed.

Accept the email-based inquiry model and maintainer-provided inquiry address model. Defer actual address publication to a dedicated contact-publication checkpoint.

## Consequences

v0.6.166 should publish the maintainer-approved inquiry address with explicit tests and claim boundaries.

This review decision does not publish an inquiry address, create customer PoC approval, create commercial contract, publish commercial license terms, approve paid engagement, approve runtime execution, approve scanner execution, approve Docker execution, approve credential use, approve customer target use, approve delivery, open an AAEF main issue, or open an AAEF main PR.
