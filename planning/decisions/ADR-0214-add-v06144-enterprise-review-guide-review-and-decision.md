# ADR-0214: Add v0.6.144 Enterprise Review Guide review and decision

Status: Accepted
Date: 2026-05-10

## Context

v0.6.143 created the Enterprise Review Guide candidate as checkpoint 1 of 2 for the Medium-risk work item selected in v0.6.142.

## Decision

Review and accept the v0.6.143 Enterprise Review Guide candidate.

The Medium-risk work item is closed. The accepted guide provides claim-safe reviewer-facing guidance for enterprise reviewers, vulnerability assessment company decision-makers, security architects, and technical reviewers.

## Consequences

Future work should select the next item using the v0.6.120 risk-tiered checkpoint granularity policy.

No runtime execution, scanner execution, Docker execution, credential injection, customer target, delivery authorization, AAEF main issue, or AAEF main PR is authorized by this review decision.
