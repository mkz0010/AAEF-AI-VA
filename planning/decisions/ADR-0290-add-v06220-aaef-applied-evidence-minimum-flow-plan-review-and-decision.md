# ADR-0290: Add v0.6.220 AAEF Applied Evidence Minimum Flow Plan Review and Decision

Status: Accepted for minimum flow planning; not applied

## Context

v0.6.219 recorded the AAEF Applied Evidence Minimum Flow Plan Candidate.

The candidate plan was created after AAEF main directed AAEF-AI-VA to reorganize existing Tool Gateway runner, mock flow, evidence chain linkage, reconstruction report, and reviewer materials into a minimum applied evidence package before adding new diagnostic capability.

## Decision

Accept the v0.6.219 candidate plan as the AAEF Applied Evidence Minimum Flow Plan for future minimum flow planning.

This decision does not create the minimum flow package or any fixture, evidence linkage, request, decision, dispatch, walkthrough, mapping, or handback artifact.
