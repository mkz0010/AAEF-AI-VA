# ADR-0266: Add v0.6.196 Repository Landing and Demo Path Clarity candidate

Status: Accepted
Date: 2026-05-13

## Context

v0.6.195 selected Repository Landing and Demo Path Clarity as a Medium-risk work item after v0.6.194 accepted the static, mock, non-execution Safe Demo Fixture Set.

## Decision

Add a draft Repository Landing and Demo Path Clarity candidate and README landing entry.

## Consequences

v0.6.197 should review the clarity candidate and decide whether to close the work item.

This candidate does not create new fixtures, public samples, schemas, validators, executable demos, runtime behavior, scanner behavior, PoC materials, or AAEF main changes.
