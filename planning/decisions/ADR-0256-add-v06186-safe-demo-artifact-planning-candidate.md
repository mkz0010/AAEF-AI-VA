# ADR-0256: Add v0.6.186 Safe Demo Artifact Planning candidate

Status: Accepted
Date: 2026-05-12

## Context

v0.6.185 selected Safe Demo Artifact Planning as a Medium-risk work item and assigned two checkpoints. v0.6.184 accepted the Safe Demo Scenario Definition.

## Decision

Add a draft Safe Demo Artifact Planning candidate.

The plan defines the artifact set that could later support the accepted Blocked Tool Action Request Review scenario, including artifact inventory, public/private artifact candidates, fixture boundaries, evidence trace boundaries, non-execution result boundaries, reviewer flow, and navigation expectations.

## Consequences

v0.6.187 should review the Safe Demo Artifact Planning candidate and decide whether to close the work item.

This candidate does not create fixtures, public samples, schemas, validators, executable demos, runtime behavior, scanner behavior, customer PoC materials, or AAEF main changes.
