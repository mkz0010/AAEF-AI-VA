# ADR-0255: Add v0.6.185 next work selection using risk-tiered granularity

Status: Accepted
Date: 2026-05-12

## Context

v0.6.184 reviewed and accepted the Safe Demo Scenario Definition candidate and closed the Medium-risk work item. The next safe step is to plan what artifact set could later support that accepted scenario without creating executable demo behavior yet.

## Decision

Select Safe Demo Artifact Planning as the next work item.

Classify the selected work item as Medium risk because it defines the plan that later demo artifacts may follow and can influence repository structure, public reviewer navigation, fixture design, evidence trace shape, and the boundary between demonstration and execution.

Use two checkpoints: candidate implementation, then review and decision.

This v0.6.185 decision itself is a Low-risk direction-selection record and is completed in one checkpoint.

## Consequences

v0.6.186 should create the Safe Demo Artifact Planning candidate and tests. No safe demo artifact is created by this v0.6.185 checkpoint.
