# ADR-0213: Add v0.6.143 Enterprise Review Guide candidate

Status: Accepted
Date: 2026-05-10

## Context

v0.6.142 selected Enterprise Review Guide as a Medium-risk work item and assigned two checkpoints.

## Decision

Add a claim-safe Enterprise Review Guide candidate for enterprise reviewers, vulnerability assessment company decision-makers, security architects, and technical reviewers.

The guide explains project positioning, review path, evidence review questions, gate-semantics review questions, demo boundaries, deployment due-diligence prompts, commercial evaluation boundaries, and conservative claim boundaries.

## Consequences

v0.6.144 should review the guide and decide whether to close the work item.

No runtime execution, scanner execution, Docker execution, credential injection, customer target, delivery authorization, AAEF main issue, or AAEF main PR is authorized by this candidate.
