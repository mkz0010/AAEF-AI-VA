# ADR-0241: Add v0.6.171 next work selection using risk-tiered granularity

Status: Accepted
Date: 2026-05-12

## Context

v0.6.170 closed the Public Entry and Inquiry Route Consistency Review work item. v0.6.120 requires future work to be selected by explicitly assigning risk tier and checkpoint count.

The maintainer clarified that AAEF reflection should be handled as a proposal text to the AAEF team, not as a direct AAEF main modification.

## Decision

Select AAEF Team Inquiry Address Reflection Proposal as the next work item.

Classify the selected work item as Medium risk because it creates proposal language about whether an AAEF-wide inquiry address accepted in AAEF-AI-VA may later be reflected toward AAEF main, while the AAEF main handback sequence remains paused.

Use two checkpoints: candidate implementation, then review and decision.

This v0.6.171 decision itself is a Low-risk direction-selection record and is completed in one checkpoint.

## Consequences

v0.6.172 should create the AAEF Team Inquiry Address Reflection Proposal candidate and tests. No proposal is created by this v0.6.171 checkpoint.
