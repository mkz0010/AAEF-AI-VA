# Exit Strategy Draft

## Strategy

Build a working MVP and validated PoC, then pursue one of the following:

- License to vulnerability assessment companies
- Joint business with MSSP / SIer
- Sale of commercial implementation
- Integration into ASM / GRC / security consulting platforms

## Assets to Preserve

AAEF itself should remain as a public framework and author-brand asset where possible.

The commercial implementation can be licensed, transferred, or sold separately.

## Assets That May Be Sold or Licensed

- Tool Gateway implementation
- Authorization Gateway implementation
- Evidence Store implementation
- Report generation pipeline
- Templates and operational workflows
- Validated PoC results
- Integration know-how
- Commercial packaging
